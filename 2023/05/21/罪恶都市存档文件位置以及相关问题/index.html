罪恶都市存档文件位置以及相关问题

2023-05-21 GTAVC, 其他, 已验证, 新手进阶 0 评论

罪恶都市游戏存档使用方法：

罪恶都市罪吧汉化版存档后缀名为sv，例如GTAVCsf1.sv

罪恶都市英文版和无名汉化存档后缀名为b，例如GTAVCsf1.b

如果是罪吧汉化版：下载好存档后，将存档的名字修改成GTAVCsf1.sv。然后放入【我的文档】里面的【GTA Vice City User Files】文件夹。进入游戏就可以读取存档了。

如果英文版和无名汉化版：下载好存档后，不需要修改名称，然后放入【我的文档】里面的【GTA Vice City User Files】文件夹。进入游戏就可以读取存档了。

注意：GTAVCsf1名称里的1可以是2，3，4，5，6，7，8。因为游戏里有八个存档，改啥名存档将出现在对应位置。

罪恶都市游戏存档所在位置：

【我的文档】里面的【GTA Vice City User Files】文件夹。

存档文件中的数字代表的是第几个存档位置，可更改存档数字范围1-8，切记不可出现重复存档，如存档中出现重复存档可能导致游戏读取不了存档内容。

更改main.scm之后原存档不可读取，在修改main.scm之后只能开始新游戏，或者读取在使用此main.scm所进行存档的存档文件，不同main.scm所存取的存档是不通用，强制读取，游戏会直接报错。

使用CLEO或者MOD会有一定几率导致存档文件损坏，存档文件损坏之后，此存档文件就不能再读取了，强制读取，游戏会直接报错。

存档文件中有个文件名为gta_vc.set的文件，为游戏设置文件，包含游戏中在设置中设置的数据，此数据为通用数据，无论电脑中有多少个罪恶都市游戏文件，只要是在读取此目录中的游戏文件除存档数据不互通，其他设置数据皆为互通使用，例如游戏分辨率，修改按键，开启/关闭HUD，开启/关闭雷达与地图，游戏音量等等。
